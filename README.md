# 项目简介
## 核心功能
- 整合销售数据
- 运用数据分析技术
- 提供可视化分析报告
- 利用Django框架搭建后端服务
- 通过MySQL数据库存储和管理数据
- 前端采用HTML结合Echarts库进行数据可视化

## 适用场景
- 农业从业者
- 销售商
- 决策者

## 技术特点
- 使用Python语言
- 基于Django框架
- 集成Echarts可视化库
- 采用MySQL数据库
- 利用HTML技术进行前端展示
